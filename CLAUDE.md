# The Challenge Web

Health goals and activity tracking PWA. Log daily health activities, view history, compete on leaderboard, track streaks.

## Tech Stack

- Nuxt 3, Vue 3, TypeScript
- Firebase backend
- Tailwind CSS
- PWA

## Commands

- `npm install` — install dependencies
- `npm run dev` — dev server on port 3000
- `npm run build` — production build
